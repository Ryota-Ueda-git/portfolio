# Portfolio

Personal portfolio site for Ryota Ueda — built with Next.js, TypeScript, and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repo at [vercel.com/new](https://vercel.com/new).
3. Deploy with default Next.js settings.
4. After the first deploy, update `site.url` in `src/content/site.ts` with your Vercel URL.

## Content updates

- Site copy: `src/content/site.ts`
- Kotobana privacy policy: `src/content/privacy.ts`

English text appears first, Japanese below throughout the site.
